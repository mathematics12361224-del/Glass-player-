Glass Table Puzzle

Hidden state · 2 touches max · secret rotation

Inspect up to two glasses, flip any inspected glass you choose, then end the turn. After each turn the table rotates by a random multiple of 90° without revealing the amount.

Click a covered glass to inspect it. That counts as one of your two touches this turn. Once inspected, click it again to flip it or leave it as is.
Fair-play rules built in:
• The start state is random, but never all-up or all-down.
• The table rotation is stored internally and applied honestly.
• The game only checks the actual hidden state, not what is visible on screen.
